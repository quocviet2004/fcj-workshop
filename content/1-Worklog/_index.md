### Internship Work Log Overview

Welcome to my **Internship Work Log**. This page was created to document and track my progress throughout a **12-week internship** (from **May 2026 to July 2026**).

The primary focus of this internship was to study and apply **Amazon Web Services (AWS)** cloud technologies together with **ReactJS** to successfully develop a real-world project: a **Mini Traveloka Booking Website** based on a **serverless architecture**.

Below is an overview of my 12-week internship journey, from learning the AWS ecosystem and designing the database to developing backend APIs and deploying the completed website to a production environment.

**Week 1:** [Introduction to AWS and its core services](1.1-week1/)

**Week 2:** [Project planning meeting and research on AWS Serverless services for the project](1.2-week2/)

**Week 3:** [Designing database tables and populating sample data in Amazon DynamoDB](1.3-week3/)

**Week 4:** [Studying the fundamentals of AWS Lambda and Amazon API Gateway](1.4-week4/)

**Week 5:** [Using AWS SDK for JavaScript v3 to develop Lambda functions for querying Amazon DynamoDB](1.5-week5/)

**Week 6:** [Configuring Amazon API Gateway, integrating it with AWS Lambda, and resolving CORS issues](1.6-week6/)

**Week 7:** [Setting up monitoring and automated notifications with Amazon CloudWatch and Amazon SNS](1.7-week7/)

**Week 8:** [Initializing the ReactJS project and implementing API calls to display dynamic data](1.8-week8/)

**Week 9:** [Enhancing the UI/UX with CSS Flexbox and refining the user interface](1.9-week9/)

**Week 10:** [Building the application and deploying the website to Amazon S3 Static Website Hosting](1.10-week10/)

**Week 11:** [Testing the application, fixing bugs, and preparing the internship final report](1.11-week11/)

**Week 12:** [Completing the internship report, obtaining the company’s official confirmation, and submitting the final report](1.12-week12/)
